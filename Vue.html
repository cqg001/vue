事件修饰符

在事件处理程序中调用 e.preventDefault和e.stopPropagation是一件常见的事情。但是原生的js写法比较麻烦难记，我们的vue框架提供了一个事件修饰符

  • .stop：阻止事件冒泡
  • .prevent：阻止默认事件发生
  • .capture：使用事件捕获模式
  • .self：只有元素自身触发事件才执行（冒泡和捕获都不执行）
  • .once：只执行一次

形式：

xxxxxxxxxx
<div v-on:click.self="函数名"></div>
<div @click.stop="函数名"></div>

示例：

xxxxxxxxxx
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8" />
        <title></title>
        <style>
            .div1{
                width: 100px;
                height: 100px;
                background-color: blue;
            }
            button{
                padding: 10px;
                outline: none;
            }
        </style>
        <script src="js/vue.js" type="text/javascript" charset="utf-8"></script>
    </head>
    <body>
        <div id="app">
            <div class="div1" @click="print('我点击了div')">
                <button @click.stop="print('我点击了button')">点击</button>
            </div>
            <a href="http://www.bilibili.com" @click.prevent="print('跳转到bilibili')">跳转到bilibili</a>
        </div>
        <script>
            var app=new Vue({
                el:'#app',
                data:{
                    
                },
                methods:{
                    print(str){
                        console.log(str)
                    }
                }
            })
        </script>
    </body>
</html>
​

小结：

  • 不使用阻止冒泡事件的时候，就没有办法阻止冒泡
  • 加入事件修饰符.stop之后，就可以阻止冒泡事件了
  • a标签有默认的点击跳转事件，使用.prevent之后可以阻止

基本语法（{‘key’:'value'}）

key值的讲解：

当列表中数据发生变化的时候，难免会出现数据的上移或者下移动。

这个时候如果我们不设置key（主键）那么会出现问题

举例

比如栈顶本身只有一个元素。我现在选择了这个元素，然后我在栈顶添加了一个元素

那么我选择的元素就会被挤到下面去，新增的元素成为了我选择的元素

计算属性的使用

目标：了解vue计算属性computed的使用

分析：通过当前获取的毫秒时间戳，来提取当前时间的出生年月日

代码：

xxxxxxxxxx
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8" />
        <title></title>
        <style></style>
        <script src="js/vue.js" type="text/javascript" charset="utf-8"></script>
    </head>
    <body>
        <div id="app">
            <!-- 使用计算属性(不需要花括号) -->
            <h2>{{getTime}}</h2>
        </div>
        <script>
            var app=new Vue({
                el:'#app',
                data:{
                    time:1609050036193
                },
                // 计算属性computed
                computed:{
                    getTime(){
                        const date=new Date(this.time)
                        return date.getFullYear()+"--"+(date.getMonth()+1)+"--"+date.getDate()
                    }
                }
            })
        </script>
    </body>
</html>

使用计算属性可以使得{{}}中的内容更简单

4.父组件向子组件通信

分析：给子组件写一个属性用来接收父组件传递的数据

代码：接收父组件给子组件传递的数组

xxxxxxxxxx
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8" />
        <title></title>
        <style></style>
        <script src="js/vue.js" type="text/javascript" charset="utf-8"></script>
    </head>
    <body>
        <div id="app">

        </div>
        <script>
            const list = {
                template: `<ul>
                <li v-for="item in items">{{item.id}}--{{item.name}}</li>
                </ul>`,
                //定义组件的属性，用来接收父组件的传递值
                props: {
                    items: {
                        //数组类型
                        type: Array,
                        //默认值
                        default: []
                    }
                }
            }
            var app = new Vue({
                el: '#app',
                data: {
                    stu: [{
                        "id": 1,
                        "name": "小程"
                    }, {
                        "id": 2,
                        "name": "小王"
                    }, {
                        "id": 3,
                        "name": "小赵"
                    }]
                },
                components: {
                    list
                }
            })
        </script>
    </body>
</html>
​

总结

  1. 组件属性props中的type可以是Object、String、Array、Number
  2. 当type:Object时候，需要注意default要写成方法:default(){return {}}
  3. v-指令对模板中根元素没有用，模板template只能写成单根树形式，不能有多个树干
  4. 子组件通过属性获取到父组件传递的数值后，不能直接改变（造成混乱）。需要用data()或者计算属性

5.子组件向父组件通信

目标：实现子组件向父组件的传值

分析：由于先有父，后有子。所以用属性传值已然无用，我们用子组件调用父组件的方法来实现反向通信

代码：点击子组件，实现父组件值的递增

xxxxxxxxxx
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8" />
        <title></title>
        <style></style>
        <script src="js/vue.js" type="text/javascript" charset="utf-8"></script>
    </head>
    <body>
        <div id="app">
            <h2>{{num}}</h2>
            <control @plus="up" @reduce="down" :snum="num"></control>
        </div>
        <script>
            const control = {
                template: `<div>
                <button @click="incrData">+</button>
                <button @click="decrData">-</button>
                </div>`,
                props: {
                    snum: {
                        type: Number,
                        default: 0
                    }
                },
                methods: {
                    incrData() {
                        return this.$emit("plus");
                    },
                    decrData() {
                        return this.$emit("reduce");
                    }
                }
            }
            const app = new Vue({
                el: '#app',
                data: {
                    num: 0
                },
                components: {
                    control
                },
                methods: {
                    up() {
                        this.num++
                    },
                    down() {
                        this.num--
                    }
                }
            })
        </script>
    </body>
</html>
​

总结：$emit()可以用来调用v-on事件，里面可以携带多个参数。携带的参数会传递给事件函数
